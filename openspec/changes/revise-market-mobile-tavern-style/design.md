- Fixed phone-preview width: keep the app around 430px max width across all tabs.
- Tavern styling:
  - dark warm brown and deep blue-black surfaces
  - gold borders and separators
  - subtle leather/wood-like gradients
  - stronger beveled buttons

- Table stays compact and no avatar images are reintroduced.
- Header and bottom navigation should feel like the provided game UI reference rather than a web admin table.

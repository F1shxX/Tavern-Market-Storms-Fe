# Asset Requirements

## UI Screens

Preferred source format:

- Figma, PSD, or Lanhu project
- PNG previews for quick review

Priority screens:

1. Home page
2. Market/ranking page
3. Market target detail page
4. Trade/support page
5. Holdings page
6. Profit/loss ranking page
7. Announcement page

Admin pages can be visually simpler in the first version.

## Image Assets

Preferred export format:

- Transparent PNG or SVG for icons and UI elements
- 2x/3x exports for mobile clarity
- English or pinyin filenames to avoid path issues

Needed assets:

- Buttons
- Icons
- Backgrounds
- Dialogs
- Ranking badges
- Virtual coin icon
- Movement indicators
- Default avatar
- Market target avatar/card images

## Avatar And Target Images

Recommended:

- PNG or JPG
- Square images, 512x512 or 1024x1024
- Original or authorized images

Do not use real streamer avatars or third-party game art unless authorization is confirmed.

## Copywriting

Needed text:

- Game rules
- Tutorial
- Entertainment-only disclaimer
- Announcement templates
- Activity descriptions
- Empty states and error messages

Avoid wording that implies real earnings, investment, recharge, withdrawal, or gambling.
